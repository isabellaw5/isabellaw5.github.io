IQ Test Project

“The Ultimate IQ Test” project is a gag-type game I designed to demonstrate a working knowledge of electronics and circuit design. The goal of the game is to "win" by following the prompts on the wooden interface pictured below. There are three levels of "difficulty", each corresponding to an intelligence level.
I have broken this page into two parts: A brief overview of the project, followed by a detailed report for those interested in the more technical mechanics of the project.

Brief Project Overview



Level One
The purpose of "Level One" one is to speak into an electret mic. To play, the user must first push the "Start" button—as seen on the interface—which activates the circuit. If the user successfully uses the microphone, the yellow LED will illuminate above "Average". The user may achieve a higher intelligence rating after completing the higher levels. This circuit demonstrates a functional knowledge of RC high pass filters, operational amplifiers, and voltage dividers.
